PSX Daily Quotes displays the quotes of companies listed on the Pakistan Stock Exchange (PSX) through a free, non-binding service provided by myself at my website https://www.adeelsarfaraz.com 

== Description ==

This plugin displays the daily quotes of all listed companies on the PSX (Pakistan Stock Exchange). 
The data is retrieved from the URL (https://www.adeelsarfaraz.com/psx/) in JSON and displayed in an HTML table on the page/post where you have placed the shortcode.
This has been provided as a free service with no strings attached.
Please go through the https://www.adeelsarfaraz.com/terms-of-service/ and https://www.adeelsarfaraz.com/privacy-policy-website/ for more details regarding usage of this plugin on your website.


== Installation ==

Upload the PSX Daily Quotes plugin to your blog, activate it and place the shortcode on the page where you wish to display the data. That's it.
